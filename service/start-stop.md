## Service Status
```sh
sudo service mysql status
```

## Start Stop Service
```sh
sudo service mysql start
sudo service mysql stop
```

## Service Disable
```sh
sudo systemctl disable mysql
sudo systemctl disable mongodb
sudo systemctl disable postgresql
```

## Service Enable
```sh
sudo systemctl enable mysql
```
